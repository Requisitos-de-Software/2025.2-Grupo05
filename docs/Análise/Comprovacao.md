# Comprovação Informal

## Introdução

Chegada na fase de validação, que é a fase de confirmação de que o produto é aquele desejado pelo usuário, uma das estratégias tomadas para o projeto foi a comprovação informal. Ela é uma descrição em linguagem natural para que os clientes possam identificar problemas na expressão dos requisitos. No caso do Bilheteria Digital, foi preferível entrar em contato com a empresa para obter a validação.

## Método de comunicação
Para a realização da validação, foi enviada uma mensagem por e-mail para a equipe do Jornada do Estudante, não foram encontrados outros meios de comunicação a não ser o e-mail oficial do aplicativo, inclusive não foram encontrados e-mails ou outros canais de conversa diretamente com a equipe de desenvolvimento do projeto.

A mensagem enviada por e-mail adotou uma abordagem estruturada e solene. A escolha desse meio de comunicação foi motivada pela necessidade de transmitir claramente informações técnicas e detalhadas relacionadas aos artefatos produzidos.
